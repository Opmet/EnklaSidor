## Installera till eclipse
(1) Välj import Projeckts from Git.

(2) Välj Clone URI.

(3) Ange https://github.com/Opmet/EnklaSidor.git

(4) Ange working directory till var/www/html/EnklaSidor.

(5) Välj Import existing projects.

(6) Tryck Finish.
